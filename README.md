# This-repo-has-N-stars

## **这个项目有n个star**

如项目名称所示，这个项目有N个Star，当star的数量发生改变时，项目名称会被动态地更新。

As the repo's name says, the repo has N stars. The name of this repo will be updated dynamically when you give it a star or unstar it.
